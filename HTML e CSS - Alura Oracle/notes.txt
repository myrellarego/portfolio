--------------------HTML-------------------------------
Documentação: W3S HTML Intro
Os <heading> (h1, h2) vão até h6
alt = texto alternativo, lê a imagem na tela

<!DOCTYPE html> 
<html>
    <head>
        <title>Portfolio</title>
    </head>
    <body>
        <h1>Isso é um titulo (h1)</h1> 
        <h2>Isso também é um titulo (h2)</h2>
        <h3>Isso também é um titulo (h3)</h3>
        <p>Isso é um paragrafo (p)</p>
        <img src="html.png" alt="logo do HTML 5">
    </body>
</html>

estrutura basica: 
    <header></header> topo da pag
    <main></main> menu
    <footer></footer> rodape

    Semantica https://developer.mozilla.org/pt-BR/docs/Glossary/Semantics
